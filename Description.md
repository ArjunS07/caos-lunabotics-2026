# Lunabotics — target hardware

- **Compute:** NVIDIA Jetson Orin Nano  
- **3D LiDAR:** Unitree L2 (top-mounted)  
- **Depth + IMU:** Intel RealSense D435i (forward)  

**Environment:** Sandy regolith, boulders, craters; prioritize avoiding craters; limited bandwidth.

**TF:** LiDAR on top, depth camera forward; static transforms in bringup should match measured CAD / mounting.
